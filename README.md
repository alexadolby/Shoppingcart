Clone or download from [GitHub source] then navigate into the project repository 

```sh
cd shoppingcart
```
## Usage 

Run the program 

```py
python shopping_cart.py
```

## Testing

Install pytest package (first time only):

```sh
pip install pytest
```

Run tests 

```sh
pytest
```
